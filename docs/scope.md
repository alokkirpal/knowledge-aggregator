# Knowledge Store Scope

## Topic
Solar System

## Competition
Science Olympiad

## Division
B

## Season
2026

## Scope Decision
The knowledge store is limited to the 2026 Science Olympiad Division B Solar System event. It is not a generic Solar System knowledge base.

## Seed Sources
1. https://www.soinc.org/solar-system-b
2. https://scioly.org/wiki/Solar_System

## Current Phase
Text-only source processing.

## Included
- HTML pages
- PDF resources
- Official text resources
- Wiki/reference text
- Practice-test text if available

## Deferred
- YouTube videos
- Video transcripts
- Image-only resources

## Ignored
- Social media links
- Store pages
- Login pages
- Unrelated links

## Reason for 2026 Scope
The Science Olympiad Solar System event focus can change every year. Therefore, sources, topics, chunks, and relevance scores must be linked to the 2026 event version.
